# fancy-cards
**Pimp your card! The fancier your card the more skilled you are.**

# Cara membuat kartu yang benar
satu orang hanya boleh membuat satu card (kartu profil). Dimana element nya harus diletakkan didalam **card parent element** khusus yang sudah memiliki style bawaan yaitu: 
```<section class="card_parent">```

## Contoh
```
<section class="card_parent">
  terserah
</section>
```

<br>
konten dalam card parent bisa dicustomize sesuka hati dengan syarat:

## Syarat
<ul>
  <li>Harus berisi nama dari pembuat</li>
  <li>boleh mengubah style bawaan dari <b>card parent element</b> kecuali height, width, dan margin</li>
</ul>

## Tips
<ul>
  <li>Desain menggambarkan personalisasi diri</li>
  <li>Memiliki efek animasi, responsive, dll</li>
  <li>Boleh ditambahkan data additional seperti umur, akun sosial media, dll</li>
  <li>Boleh memodifikasi style diluar card parent. Jika style nya memang bagus nanti akan di publish</li>
</ul>
